We propose a model-driven approach for enforcing fine-grained access control (FGAC) in SQL queries using a function that generates secure stored procedures. To improve performance, we introduce a model-based optimization methodology that removes unnecessary authorization checks. Using a mapping from OCL to many-sorted first-order logic, SMT solvers can prove when such checks are redundant. A case study demonstrates the applicability of the approach.
